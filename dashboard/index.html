How it works

The classifier is built with scikit-learn using a TF-IDF representation and a logistic regression model. It was trained on curated examples of phishing and legitimate corporate emails. Confidence scores show how strongly the model believes an email belongs to a class; always combine model results with security best practices.

Run the training pipeline with python ml/train_model.py before starting the API.
