Train the PhishGuard Model

PhishGuard learns from curated examples of phishing and legitimate communication. The workflow below guides you through running the training pipeline and evaluating the results.

Try the live predictor →

Training workflow

  1. Prepare the environment: python3 -m venv .venv && source .venv/bin/activate && pip install -r requirements.txt
  2. Run the training script: python ml/train_model.py
  3. Inspect generated artifacts in ml/:
    • phishguard_model.joblib – serialized model pipeline
    • training_metrics.json – accuracy, precision, recall, and F1
    • classification_report.txt – class-level metrics
    • confusion_matrix.png – model performance visualization
  4. Launch the API: uvicorn api.main:APP --reload

Dataset snapshot
